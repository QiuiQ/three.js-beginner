# three.js-beginner
## Installation
node.js (my version is [12.18.3]) to use npm
```bash
npm install --global http-server
npm install --save three
```
Notice:create a folder to put three.js tools and modules

## http-server command
```bash
http-server -c-1 -p 8010
```
